# Student Depression Data Analysis – SQL + Tableau Project
##   Project Description
This project focuses on Analysing student depression data using SQL Server and Tableau to understand the impact of academic, financial, and lifestyle factors on students’ mental health. The analysis combines data preparation in SQL with interactive visual analytics in Tableau, resulting in a clear, insightful dashboard suitable for decision-makers in education and mental health domains.

##   Tools & Technologies
•	Microsoft SQL Server
•	SQL (Data Cleaning & Transformation)
•	Tableau Desktop
•	Tableau Cloud

##   Project Implementation Steps
1️⃣ Data Preparation in SQL Server
•	Imported raw student depression data into SQL Server
•	Cleaned and standardized the Gender column
•	Created an Age Group column for demographic analysis
•	Analysed column distributions to understand data spread
•	Added an Index column and updated the Depression indicator for consistency

2️⃣ Tableau Integration
•	Installed Tableau Desktop
•	Connected Tableau directly to SQL Server
•	Verified data accuracy after integration

3️⃣ Analytical Visualizations
Created student count analysis based on key factors:
•	Academic Pressure vs Student Count
•	Financial Stress vs Student Count
•	Study Satisfaction vs Student Count
•	Sleep Duration vs Student Count
•	Study Hours vs Student Count

4️⃣ Dashboard Creation
•	Designed a Student Count Analysis Dashboard
•	Combined multiple visuals for holistic insights
•	Applied filters for interactive exploration

5️⃣ Publishing & Sharing
•	Published the final dashboard to Tableau Cloud
•	Enabled secure access for stakeholders

##   Key Insights Delivered
•	Identified relationships between academic pressure and depression levels
•	Highlighted impact of financial stress on student mental health
•	Analysed how sleep duration and study hours correlate with depression
•	Provided an interactive dashboard for deeper exploration

##  Conclusion
This project demonstrates an end-to-end analytics workflow using SQL for data transformation and Tableau for visualization. It highlights the ability to turn raw mental health data into meaningful, actionable insights through clean data Modeling and well-designed dashboards.
